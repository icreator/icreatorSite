Программа для расчета коммунальных платежей для товарищества собственников жилья (ТСЖ и ЖКХ).

Программа позволяет рассчитывать коммунальные платежи для ЖКХ: ТСЖ или ЖЭУ, печатать квитанции для оплаты коммунальных услуг, просматривать должников и другое. Особенность программы состоит в том что можно вести учет по долям собственности на одну квартиру. То есть в одной квартире может быть несколько собственников, на каждого из которых заводится отдельный лицевой счет. Для жилищно-коммунальных хозяйств программа по расчету коммунальных услуг, и печатание квиточков оплаты ЖКХ - это тоже самое.
