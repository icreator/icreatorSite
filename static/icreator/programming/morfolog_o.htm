    Я студент Евгений, хочу задать к вам вопрос по алгоритму компьютерного морфологического разбора русских слов. Очень не понятно мне обьяснение на Прологе нельзя ли как нибудь по-другому. Просто мне для курсового проекта требуется написать программу на С++ для вывода из БД по заданному слову все его словоформы. Подскажите пожалуйста как проще если можете. Заранее благодарен.

А почему на С++ - ведь это чисто задача для СУБД. Возьмите хоть ACCESS.
Суть алгортма полностью описана в статье:
----------------------------
Морфология слов русского языка определяется по аффиксу...
---------------------------
если Вам нужен С++, алгоритм останется тот же, хотя нет, для Вас немного другой:
1. введенное слово найти в базе данных.
2. у этого слова взять основную начальную форму . пример. для "стулья - стул", для "крассным - красный", для "купалися - купаться"
3. ищем в базе основную начальную форму.
4. берем у нее все нужные словоформы.
Здесь структура базы данных будет следующей:
таблица Словарь
Слово, ОсновноеСлово
таблица Словоформы
ОсновноеСлово, список:Словоформы.

Вообще, если писать на СУБД, то можно сделать реляционную связь между Словарь:ОсновноеСлово и Словоформы:ОсновноеСлово. Тогда достаточно сделать поик введеной строки в Словарь:Слово, после чего взять значения Словоформы:Словоформы.
Замечу, что ели в поле Словоформы нельзя вводить список в данном языке программирования, то нужно сделать еще одну базу и установить реляционную связь с ней.
