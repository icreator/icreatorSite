Тепловая энергопроводность ( Ер/с/К = Дж/К2/с = Вт/К2 = (Ер/с)2/Вт)

ёмкость потенциальной энергии, энергоёмкость (с*ТОК/ПОТ = ЗАР/ПОТ = ЭНЕ/ПОТ2)

C= I*dt/dj = dX/dj = X/j =2*EC/j2

CT= IT*dt/dT = dXT/dT = XT/T =2*ECT/T2

Тепловая энергоемкость (Ер/К = Дж/К2)

ёмкость кинетической энергии, энергоинертность (НАП*с/ТОК = НАП*с2/ЗАР2 = ЭНЕ/ТОК2)

L =Y*dt/dI =2*EL/(X/t)2 =Y/a = Y*t2/dX

LT =DT*dt/dIT =2*ELT/(XT/t)2 =DT/aT = DT*t2/dXT

Тепловая энергоинертность (К*с2/Ер = Дж/(Ер/с)2). Отражает скорость передачи тепловой энергии при теплопроводности

Потенциальная энергия (может быть только накапливаемой)  (ЭНЕ)

EC= C*j2/2,

dEC= X*dj = dEY

ECT= CT*T2/2,

dECT= XT*dT = dEYT

Накапливаемая тепловая энергия, внутренняя тепловая энергия (Дж)

Кинетическая энергия (может быть только накапливаемой) (ЭНЕ)

EL= L*(dX/dt)2/2 = L*I2/2

ELT= LT*(dXT/dt)2/2 = LT*IT2/2

Энергия теплового удара (Дж)

Энергия сопротивления, энергия рассевания. Эта энергия является передаваемой. (ЭНЕ)

ER= Y*X=R*X2 = t*I2*R = t*Y2/R

ERT= DT*XT=RT*XT2 = t*IT2*RT = t*DT2/RT
